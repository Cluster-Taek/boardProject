# BoardProject

## 개발 목표
spring4.0을 활용하여 게시판을 제작하고 CRUD 구현을 통한 spring 프레임워크 이해

## Tools
* 통합개발환경 :	Eclipse 2020-06
* 개발 도구 :	JDK 1.8﻿
* 메인 프레임워크 :	Spring4.3.8
* 빌드 :	Maven
* 데이터베이스 연동	: myBatis
* 데이터베이스	: MariaDB 10.5
* UI	 : Bootstrap 4
